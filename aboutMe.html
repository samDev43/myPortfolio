About Me

Hi, I’m Sooto Olalekan Samad a curious learner and problem solver from Nigeria. I love creating things on the web, from clean and responsive designs to interactive apps. I started with HTML and CSS, moved into JavaScript, and now I’m exploring PHP and backend development. I enjoy building projects that teach me new skills, like secure login systems and fun quiz apps.

I’m also passionate about cybersecurity. Right now, I’m learning how to test and protect web applications, using tools like Kali Linux and practicing ethical hacking techniques.

When I’m not coding, I like boxing and finding simple ways to explain tech ideas to others.

My goal is to grow as a full-stack developer and cybersecurity expert, creating solutions that are both useful and safe for everyone.
